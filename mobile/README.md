# FaithConnect Mobile App

React Native application for iOS and Android.

## Tech Stack

- React Native 0.72+
- Redux Toolkit
- React Navigation 6
- WatermelonDB (offline-first)
- Socket.io-client

## Setup

```bash
# Install dependencies
npm install

# iOS
cd ios && pod install && cd ..
npm run ios

# Android
npm run android
```

## Features

- Phone OTP authentication
- Social feed
- Live prayer meetings
- Bible reader
- Quiet time journal
- Direct messaging
- Groups
