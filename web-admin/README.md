# Web Admin Dashboard

Next.js admin dashboard for content moderation, user management, and analytics.

## Tech Stack

- Next.js 14+
- TypeScript
- React Query
- Role-based access control

## Setup

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build
npm run build
```

## Features

- Content moderation queue
- User management
- Creator review
- Analytics dashboard
- Audit logs
